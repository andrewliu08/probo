<!-- GETTING STARTED -->
## Getting Started
1. Clone the repo
2. `cd frontend/`, install all required NPM packages
   ```sh
   npm install
   ```
3. follow frontend/ README instructions to start frontend website
4. `cd backend/` and follow README instructions to start backend functionalities
